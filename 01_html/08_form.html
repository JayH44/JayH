HTML 폼데이터

  • form: 서버에서 필요로 하는 데이터를 사용자로부터 입력받아 전송하는 태그
    • method를 get 으로 지정하면 데이터 url에 포함되고 post는 포함되지 않는다.
    • novalidate 유효성 검사를 생략한다.
  • input: 사용자로부터 데이터를 입력받는 태그
    • 닫는 태그가 필요없다.
    • type 속성을 통해서 다양한 형태의 데이터를 입력할 수 있다.
      • text
      • password
      • number: min, max 설정가능
      • checkbox
      • radio: 여러 옵션중 하나만 선택할 수 있다. (name 값이 같은 타입)
    • action 속성은 데이터를 제출할 서버를 지정한다.
    • name은 서버에 전송하는 데이터의 이름
    • required로 필수 입력값을 지정할 수 잇다.
    • readonly데이터를 읽는것만 가능하다.
    • disabled항목을 비활성화 한다.

  • label: input 과 글씨를 연결할 수 있다.
    • 글씨를 클릭하면 해당 input을 클릭한 것과 동일한 효과
    • 1) 글씨를 input 과 함께 label 태그로 감싼다.
    • 2) input id 값을 지정하고 label for값을 input 의 id 로 지정한다.

  • button: form 안에 버튼을 작성하면 특정 기능을 수행한다.
    • type 에 따라서 다른 동작을 한다.
    • type 의 기본값은 submit으로 데이터를 서버에 전송한다.
    • type reset은 form 내의 모든 input 값을 초기화 한다.
    • button form 속성에 form 태그의 id 값을 지정하면 태그 밖에 있어도 동작한다.

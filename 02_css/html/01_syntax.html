CSS 적용 방법

  1. 인라인 방식

    태그에 style 속성을 사용해서 지정한다. 일일히 지정해서 사용하는 것은 가독성이 떨어지고 재사용성과 유지보수가 어렵기 때문에 잘 사용하지 않는다.

  2. style 태그 방식

    style 태그안에 css 문법에 맞게 스타일을 지정한다. 인라인 양식보다는 낫지만 style 태그 역시 html 문서가 길어지고, 재사용성도 떨어져서 잘 사용하지 않는다.

  3. 외부 파일 연결 방식

    외부 파일로 .css 확장자로 작성한다. link 태그를 통해서 css 파일을 연결한다.

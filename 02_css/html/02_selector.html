CSS 선택자

CSS 선택자란 스타일을 적용하고자 하는 요소를 골라내는 것이다.

선택자의 종류

  • 전채 선택자(*)

    전체 선택자는 html 모든 요소에 적용된다. 우선순위 점수 0점 이다.

  • 타입 선택자

    태그명을 통해서 요소를 선택한다. 문서내에 있는 해당하는 타입의 요소를 모두 선택한다. 우선순위 점수 1점 이다.

  • 클래스 선택자(.)

    특정 클래스값을 가진 요소를 모두 선택한다. 클래스 값은 요소의 class 속성을 통해서 지정한다.
    띄어쓰기를 통해서 여러개의 클래스를 지정할 수 있다.
    같은 클래스 값을 여러 요소가 가질수 있다.
    우선순위 10점 이다.

  • 아이디 선택자 (#)

    특정 id 값을 가진 요소를 선택한다. 아이디 값은 요소의 id 속성을 통해서 지정한다.
    아이디 값은 여러개를 지정할 수 없다.
    문서내의 아이디값은 유일해야 한다.
    우선순위 점수는 100점이다.

  • 일치선택자

    여러선택자에 모두 일치하는 요소만을 선택한다.

  • 자식선택자(>)

    특정요소의 자식 요소만을 선택할 수 있다.

  • 후손 선택자

    특정 요소의 후손 요소를 모두 선택할 수 있다. 띄어쓰기를 통해서 구분한다.

  • 형제 선택자 (~, +)

  • 가상 클래스 선택자

    특정 조건을 만족했을때(ex 요소에 마우스가 올라갔을때) 가상의 클래스가 생긴 것처럼 동작한다.
    hover, active, nth-child()

  • 가상 요소

    html 문서에 포함되지 않았지만 디자인적으로 필요할 대 가상으로 요소를 만들어낸다.

상속과 우선순위

글씨와 관련된 속성은 후손 요소에게도 상속된다.

같은 속성에서 값을 여러개를 적용하면 기본적으로 마지막에 적용된 값이 적용된다.
인라인 방식은 우선순위 점수가 1000점이기 때문에 주의해야한다.
!important 가 가장 최우선 순위이다.
점수가 같을때는 마지막에 작성한 코드가 적용이 된다.

!important > 인라인 > (#) id 선택자 > (.) 클래스 선택자 > 태그 선택자 > (*) 전체선택자
